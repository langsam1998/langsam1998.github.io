链表中倒数第k个节点
发表于 2020-04-07| 更新于 2020-04-07|编程题
阅读量:|评论数:

一、题目

1、题目描述

输入一个链表，输出该链表中倒数第k个节点。为了符合大多数人的习惯，本题从1开始计数，即链表的尾节点是倒数第1个节点。例如，一个链表有6个节点，从头节点开始，它们的值依次是1、2、3、4、5、6。这个链表的倒数第3个节点是值为4的节点。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/lian-biao-zhong-dao-shu-di-kge-jie-dian-lcof

二、解题思路

1、思路一

链表倒序，很容易想到栈，先进后出的特点，把链表全部压入栈内，然后根据k取出想要的值。这个思路发现执行时间比较长。

执行用时 :1 ms, 在所有 Java 提交中击败了29.64%的用户

内存消耗 :38 MB, 在所有 Java 提交中击败了100.00%的用户

2、思路二

这个题想要得到倒数第k个节点，由于是链表，必须找到最后一个节点或者找出链表长度，才能知道哪个节点是倒数第k个。

这里可以使用快慢指针，快慢指针在链表里很实用，找中间值，判断链表中的环等。

两个指针从头节点开始，找倒数第k个，让前指针比后指针先走k步，然后两个指针同时前进，当前指针走到尾结点时，后指针就在倒数k+1位置上。
