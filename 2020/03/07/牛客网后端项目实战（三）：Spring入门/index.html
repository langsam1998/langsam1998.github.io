牛客网后端项目实战（三）：Spring入门

Spring全家桶

  • Spring Framework：基石
  • Spring Boot：build anything
  • Spring Cloud：coordinate anything
  • Spring Cloud Data Flow：connect anything

Spring Framework

  • Spring Core
    • IoC、AOP（IoC控制反转，AOP面向切面，是Spring的核心，是管理bean的思想）
  • Spring Data Access
    • Transactions、Spring MyBatis（数据库事务和mybatis）
  • Web Servlet
    • Spring MVC（web服务）
  • Integration
    • Email、Scheduling、AMQP、Security（邮件，定时任务，消息队列，安全服务）
