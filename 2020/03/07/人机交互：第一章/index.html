人机交互：第一章

阅读量:

Keywords

  • 人机交互HCI（Human-Computer Interaction）
  • 图形用户界面GUI（Graphical User Interface）
  • 用户中心设计（User Centered Design）
  • 用户体验（User Experience）
  • 易用性（Usability）

人机交互HCI

是什么：人机交互是一个跨学科领域，计算机科学家，工程师，心理学家，社会学家和专业设计师扮演重要角色。是一个涉及“用户使用的交互式计算机系统”的设计，评估，实现以及相关主要现象研究的学科。

解决什么：HCI解决实际问题中的设计，技术选用以及使计算机系统对使用人员和组织更简便高效。

用户界面

是什么：用户界面是指人与机器交互发生的地方。

需要什么：UI需要高效的操作和对机器的控制，以及帮助操作人员抉择的反馈机制。

哪些是：命令行界面、图形用户界面，触控用户界面，声控用户界面

图形用户界面

是什么：GUI就是一种基于图形而不是纯文字的用户界面。

各种操作通常直接通过操纵图形元素。

易用性

是什么：用户能够多好的使用系统功能

  • 易学性Learnability
  • 高效性Efficiency
  • 可记忆性Memorability
  • 错误少而可修复
  • 满意度Satisfaction

不用用户对易用性维度的需求不同，且易用性也是只软件设计者考虑的一个方面

Functionality Usability Performance Size Cost Reliability Security Standards

许多设计抉择都要权衡不同特性。

用户体验

是什么：用户使用产品、系统或者服务的感受

它强调人机交互和产品所有权的经验、情感、意义和价值方面。它包含了一个人对系统实用性易用性和高效性的实际认知。

用户为中心的设计

用户界面影响对软件的感觉，好的设计应该表现得如用户期待的一样。

UI应该与用户交流而不是其它工程师，用户永远是对的但他们不是设计者。
